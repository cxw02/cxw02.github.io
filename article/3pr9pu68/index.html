1. 安装Termux

点击这里打开Termux的github仓库

找到

点击Release，找到

点击下载，然后在手机上安装软件。

2. 设置Termux

2.1 安装常用软件

pkg install nmap
pkg install tree

2.2 设置访问本机存储

打开Termux终端窗口，输入以下命令：

termux-setup-storage

此时系统会弹出一个权限请求框，点击”允许“即可。 授权完成后就可以使用cd命令切换到存储目录，并使用ls命令查看存储目录下的文件和文件夹。

注意

特别注意：千万要小心不要删除或更改任何关键文件，以免对系统造成损害。

cd /sdcard/

2.3 开启SSH服务

在Termux中开启SSH服务后就可以使用其他设备通过SSH协议连接到你的Android设备，从而在移动设备上执行各种命令和操作。 安装openssh软件包，在Termux终端输入以下命令：

安装：pkg install openssh
启动 ssh 服务：sshd
如果报错：sshd: no hostkeys available -- exiting.
解决方式：ssh-keygen -A

获取用户名和密码：

获取用户名：whoami 
设置新密码：passwd
默认sshd端口：8022
获取ip：ifconfig

使用SSH协议连接到Android设备的时候，使用以下命令连接到Android设备：

ssh <name>@<Android ip> -p 8022

2.4 关闭SSH服务

一旦你在Termux中使用sshd启动了SSH服务，你就会发现，每次打开Termux后，SSH服务都会自动启动。 解决方法也过于简单。只需要使用

pkill sshd

即可立刻关闭SSH服务。 想验证SSH务是否已关闭，只需要检查本机的SSH端口是否已经关闭。

nmap localhost -p <端口>
