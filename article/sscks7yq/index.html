git清理记录

约 192 字小于 1 分钟

2025-04-04

1. 恢复某个版本（先用git status查看是否改动）

git log
git reset --hard 76eac54

2. 删除所有Commit提交记录

（1）创建孤立分支，并切换到该分支：

git checkout --orphan latest_branch

（2）暂存所有文件：

git add -A

（3）提交所有更改：

git commit -am "First Commit"

（4）删除主分支 main：

git branch -D main

（5）重命名当前分支为 main：

git branch -m main

（6）强制推送本地分支：

git push -f origin main

3. 删除旧的reflog条目

git reflog expire --expire=90.days.ago --expire-unreachable=now --all
