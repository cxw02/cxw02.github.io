构建项目

  1. 设置正确的 base 选项。
.vuepress/config.ts
export default defineUserConfig({
  host: '0.0.0.0',
  port: 8080,
  base: '/', 
  lang: 'zh-CN',
  title: 'My Vuepress Site',
})
  1. 构建生产环境
npm run docs:build

部署项目

本地部署

  1. 打开控制面板→程序和功能→启用或关闭Windows功能

  2. 勾选 Internet Information Services 和 Internet Information Services 可承载的Web核心

  3. 重启电脑后打开 Internet Information Services (IIS)管理器
